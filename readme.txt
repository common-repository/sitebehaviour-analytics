Easily integrate tracking codes into your website for visitor behavior analytics with SiteBehaviour Analytics.

== Description ==

SiteBehaviour Analytics is a simple WordPress plugin that allows you to add your tracking code directly into the <head> section of your website for seamless visitor tracking. 
This tool is designed to integrate with SiteBehaviour Analytics, your one-stop solution for website analytics. Once you sign up for an SiteBehaviour Analytics account, you'll receive a unique tracking code that you can paste into the plugin settings.

= Features =
- Add a custom tracking code to the <head> section of your website
- Simple interface for pasting and managing your analytics tracking code
- Integrates with your SiteBehaviour Analytics account for real-time visitor data
- Lightweight and optimized for performance

= How it works =
1. Sign up for an SiteBehaviour Analytics account at https://web.sitebehaviour.com.
2. Once you've created an account, you'll get a unique tracking code in your workspace's website settings.
3. Paste the tracking code into the SiteBehaviour Analytics plugin settings under **Settings > SiteBehaviour Analytics**.
4. Save your changes, and you're done! Your website is now being tracked through SiteBehaviour Analytics.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/SiteBehaviour-analytics/` directory, or install the plugin through the WordPress Plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Settings > SiteBehaviour Analytics** to configure the plugin.
4. Paste your SiteBehaviour Analytics tracking code and save the settings.

== Frequently Asked Questions ==

= How do I sign up for an SiteBehaviour Analytics account? =
You can sign up for an SiteBehaviour Analytics account by visiting https://web.sitebehaviour.com.

= Where do I find my tracking code? =
After signing up and logging into your SiteBehaviour Analytics account, you'll find your tracking code in the Workspace section .

= Will SiteBehaviour Analytics slow down my website? =
No, SiteBehaviour Analytics is optimized to be lightweight and adds only a minimal amount of JavaScript to your website.

= Can I remove the tracking code from my website? =
Yes, if you want to stop tracking, simply remove the tracking code from the plugin settings and deactivate or uninstall the plugin.
